# Task: Convert Markdown Yearly README.md to report.pdf

**Issue:** #21  
**Branch:** `feat/convert-markdown-readme-to-report-pdf`

## Context

After running `unified_readme.py`, each year directory contains a `README.md` with the
exact values to enter on French tax forms. Users need a printable PDF version of this
document.

## Feature description

New standalone script `src/md_to_pdf.py`:

- Reads `<ifu-root>/<year>/README.md` generated by `unified_readme.py`
- Produces `<ifu-root>/<year>/report.pdf` in the same directory
- Converts Markdown → HTML (via `markdown` package, `tables` extension)
- Renders HTML → PDF via a headless Edge or Chrome browser (`--print-to-pdf`)
- Writes HTML to a temp file, invokes the browser, then cleans up
- Preprocesses `✓` → `Oui` and `⚠` → `(!)` to stay safe with any font
- Exits with error and a clear message if the README does not exist
- Requires Edge or Chrome to be installed (Edge is always present on Windows 11)

## Dependencies

- `markdown` — Markdown-to-HTML conversion (added to `requirements.txt`)
- No additional Python package needed; PDF rendering is handled by the system browser

## Usage

```bash
python src/md_to_pdf.py 2024
python src/md_to_pdf.py 2024 --ifu-root ifu
```
